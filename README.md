Интерфейс для работы с EasyOCR, включающий обученные модели для якутского и юкагирского языков, а также возможность корректировки слов через PySpellChecker.

## Установка

Требуется Python 3.10 и выше:

```bash
pip install -r requirements.txt
```

## Запуск

Осуществляется скриптом запуска `run.pyw` через Python.

## Инструкция

Приложение работает в двух режимах: работа с изображениями (`jpeg`, `png`) и работа с PDF. Данные для работы с файлами должны храниться в папках `data` и `pdf_data` соответственно.

Для загрузки новых файлов во время работы программы необходимо поместить их в соответствующие папки и нажать кнопку **Обновить**. Для выбора файла для OCR следует выделить его в списке и нажать кнопку **Выбрать** — выбранное изображение отобразится в левой части окна, при этом поддерживается приближение, отдаление и перемещение изображения.

Во вкладке **Настройки** можно выбрать доступные модели OCR, язык для PySpellChecker, а также символы для распознавания. Кнопка **Генерация** запускает обработку выбранного изображения, и сгенерированный текст появляется в правой части окна. С помощью кнопки **Понизить буквы** можно корректировать неправильно распознанные буквы по строчности. Кнопка **Выделить некорр.** подсвечивает слова, которые не соответствуют словарю, и активирует режим исправления слов. При нажатии правой кнопки мыши на выделенное слово появляется контекстное меню с вариантами исправления. Кнопка **Очистить** возвращает текст в исходное состояние и отключает режим исправления слов.
